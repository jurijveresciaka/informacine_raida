1. Užduotis C#

Parašykite biblioteką (dll), kuri turėtų viešai pasiekiamą metodą GetNaktinisLaikas(DateTime, DateTime, DateTime, DateTime), kuris leistų apskaičiuoti kiek minučių iš duotojo laikotarpio sudaro naktinis laikas, kai yra žinomos keturios datos:

  • Laikotarpio pradžios data (pvz: 2013.01.01 00:00:00)
  • Laikotarpio pabaigos data (pvz: 2013.01.01 23:50:00)
  • Naktinio laiko pradžia (pvz: 00:00:00, šiuo atveju data nesvarbi)
  • Naktinio laiko pabaiga (pvz: 05:00:00, šiuo atveju data nesvarbi)
